# Ecommerce Churn Analysis

A Machine Learning based web application that predicts customer churn and provides analytics for customer retention.

## Features

- Customer churn prediction
- Customer analytics dashboard
- Interactive Streamlit interface
- Feedback collection system
- SQLite database integration

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- SQLite
